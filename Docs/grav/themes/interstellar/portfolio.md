## Introduction

The **Portfolio** example page demonstrates how you can create a beautiful page with the Interstellar theme. Here is some information to help you replicate this page as it appears in the demo.

## Widgets and Particles

Below is a brief rundown of the widgets and particles used to make up the demo page.

![](assets/page_portfolio.jpeg)

:   1. **Slideshow** Simple Content (particle) [6%, 45%, se]
	2. **Feature** Info List (particle) [15%, 8%, se]
	3. **Feature** Image Grid (particle) [48%, 8%, se]
	4. **Extension** Info List (particle) [74%, 8%, se]
